Dapatkan Pemberitahuan Jadwal Bulan Ramadhan 1443H


Cara Penggunaan Aplikasi

  1. Jadwal Maghrib/Imsak

    2. Jadwal akan tersedia disini dan dan dapatkan notifikasi di hpmu dengan mengaktifkan Notic

  2. Kartu

    3. Gunakan Fitur ini untuk membuat Kartu Ucapan Ramdhan dan dapatkan feedback dari orang lain

  3. Catatan Ramadhan

    4. Gunakan aplikasi ini untuk mencatat kegiatan ramadhanmu dan akan tersimpan Local Storage Web Browsermu aplikasi ini bisa di jalankan secara offline tanpa harus online, jadi jangan Clear chache dan Data Pada Browsermu jika hal ini dilakukan maka seluruh data yang diinputkan akan secara otomatis terhapus.

  4. Al-Qur'an

    5. Fitur ini adalah fitur E-Al Qur'an yang digunakan untuk membaca Al-Quran secara Online. Di ambil dari API Al-Quran-JSON-Indonesia-Kemenag
